# Image encryption

Encrypt an image with a graphic implementation of **one time pad**.

## Usage

Encrypt and store an _image_:

```bash
python3 encrypt.py <input_file>
```

Show a decrypted _image_:

```bash
python3 encrypt.py <key_file> <crypto_file>
```
